# api

Bun + Hono API app template (used by `@merossity/new`).

## Dev

```bash
bun run dev
```

## Production

```bash
bun start
```

## Test

```bash
bun run test
```

## QA

If you're using this template inside the Merossity monorepo:

```bash
bun run --cwd packages/qa qa:init --dir . --kind lib
```
